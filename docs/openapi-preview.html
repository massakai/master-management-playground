Product Category Master Management API

商品カテゴリマスター管理システム MVP の API 定義。 対象機能は商品カテゴリ一覧取得、CSV ダウンロード、CSV アップロードによる全件入れ替え更新とする。

OpenAPI 3.0.3 / version 1.0.0

Endpoints

get /api/categories

商品カテゴリ一覧取得

display_order 昇順、同順位は category_code 昇順で商品カテゴリ一覧を返す。

operationIdlistCategories
requestBody-
responses
  • 200 正常
  • 500
get /api/categories/csv

商品カテゴリ CSV ダウンロード

現在のマスター内容を UTF-8 の CSV として返す。

operationIddownloadCategoriesCsv
requestBody-
responses
  • 200 正常
  • 500
post /api/categories/csv

商品カテゴリ CSV アップロード

CSV を検証し、正常時は商品カテゴリマスターを全件入れ替え更新する。

operationIduploadCategoriesCsv
requestBodymultipart/form-data
responses
  • 200 更新成功
  • 400 入力エラーまたは CSV バリデーションエラー
  • 500
